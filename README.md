# GamesUp

School case study: implementation of a Java/Spring API representing the backend of a board games shop, backed by a MySQL database and a Python machine learning API that generates game recommendations.

## Project Overview

GamesUp is a three-component backend system:

| Component           | Technology | Port |
|---------------------|---|---|
| Java REST API | Spring Boot | `8080` |
| Database            | MySQL | `3306` |
| Recommendation API  | FastAPI (Python) | `8000` |

---

## Java / Spring API

### Prerequisites

- Java 21+
- Maven
- A running instance of MySQL
- A `.env` file at the root of the project (a dependency is included in the project to read it)

### Environment variables

Create a `.env` file at the project root with the following variables:

```env
DB_USERNAME=your_mysql_username
DB_PASSWORD=your_mysql_password
JWT_SECRET=your_jwt_secret
JWT_ISSUER=com.gamesUP
```

> `RECOMMENDATION_API_URL` defaults to `http://localhost:8000` if not set.

### Run

```bash
mvn spring-boot:run
```

The API will be running on port 8080. 
API documentation is available at http://localhost:8080/swagger-ui/index.html.

### Tests

```bash
mvn test
```

Find a coverage report generated by Jacoco at  `target/site/jacoco/index.html`.

## MySQL Database

### Prerequisites

- MySQL server running locally on port `3306`
- A database named `GamesUP` created before starting the Spring API

### Seed data (optional)

A seed script is available to populate the database with initial data, run it with your MySQL credentials :

```bash
bash DB_USERNAME=your_mysql_username DB_PASSWORD=your_mysql_password ./scripts/seed-db.sh
```

## Python Recommendation API

The recommendation engine is a FastAPI application using a scikit-learn ML model trained on purchase history.

### Prerequisites

- Python 3.14+

### Setup and run

```bash
# Create and activate a virtual environment
python3 -m venv venv
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Start the API server
uvicorn main:app --reload --port 8000
```

> The Python API is only needed for the recommendation service.
